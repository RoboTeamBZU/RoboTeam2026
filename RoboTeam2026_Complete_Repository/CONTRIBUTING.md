# Contribution Guidelines

## Commit Messages

Use clear messages such as:

- `Add Version 1 chassis documentation`
- `Update LiDAR wiring diagram`
- `Test 350 RPM motor under load`
- `Improve obstacle path-planning logic`
- `Add parking calibration results`

Avoid messages such as:

- `update`
- `changes`
- `final`
- `fix`

## Before Committing

- verify filenames,
- remove duplicated files,
- include units,
- update links,
- and check Markdown preview.

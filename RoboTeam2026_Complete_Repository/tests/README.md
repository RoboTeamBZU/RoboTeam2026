# Testing and Validation

Testing must show how engineering decisions were made.

## Standard Test Record

For every test, include:

- date,
- robot version,
- field configuration,
- software version or commit,
- battery voltage,
- test procedure,
- success metric,
- result,
- observations,
- video or photo evidence,
- and decision.

## Recommended Metrics

- successful laps out of total attempts,
- average lap time,
- standard deviation of lap time,
- pillar-side accuracy,
- number of wall contacts,
- localization error,
- parking success rate,
- peak current,
- and maximum component temperature.
